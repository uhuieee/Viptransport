# Viptransport
Viptransport — это адаптивный сайт для заказа премиальных трансферов, включающий услуги перевозки, страницы автопарка, контактов и информации о компании.
Проект выполнен на HTML/CSS/JS, ориентирован на минимализм, удобство и быструю загрузку.Главная страница — описание услуг, преимущества сервиса
Страница “Автопарк” — бизнес-класс, седаны, минивэны
Страница “Сервисы” — трансферы, аренда авто с водителем, экскурсии
Контакты — номера телефонов, социальные сети, форма связи
О компании — информация о бренде и услугах
Полностью адаптивный дизайн (ПК, планшеты, мобильные устройства)
